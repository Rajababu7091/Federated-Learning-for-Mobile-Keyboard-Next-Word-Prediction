TECHNICAL DETAILS
================================================================================

Architecture:
  - Attention-GRU model
  - Embedding: 96 dimensions
  - GRU: 192 units
  - Attention: 64 units (Bahdanau)
  - Total parameters: 332,916

Federated Learning:
  - FedProx (proximal term μ=0.01)
  - FedNova (normalized averaging)
  - 25 rounds, 4 local epochs
  - Batch size: 16
  - Cosine LR decay

Privacy:
  - DP-SGD (gradient clipping + noise)
  - Clip norm: 1.0
  - Noise multiplier: 0.1
  - RDP accountant for privacy tracking

Prediction:

================================================================================
TROUBLESHOOTING
================================================================================

If predictions don't work:
  → Check model badge in sidebar (should be green "Model Ready")
  → If red, go to Train tab and click "Train Model"
  → Wait for training to complete
  → Return to Keyboard tab

If port 3000 is busy:
  → Edit app.py: change port=3000 to port=3001
  → Access http://localhost:3001

If page doesn't load:
  → Check server is running (see startup messages)
  → Try clearing browser cache (Ctrl+Shift+R)
  → Check firewall isn't blocking port 3000
